## ✅ Sections Included

- **Home** — Photo, name, intro with typing animation
- **About Me** — Education, interests, goals, strengths, skills
- **Achievements** — Ai4A, Design Thinking, ACE projects + Coursera, NetAcad, Yuva AI certs
- **Projects** — Screenshots, descriptions, GitHub & live links
- **Gallery** — Lightbox-enabled photo & certificate gallery
- **Contact** — Email, phone, LinkedIn, GitHub + contact form

---

## 🚀 How to Deploy on GitHub Pages

1. Create a new GitHub repository named `portfolio`
2. Upload all files maintaining the folder structure above
3. Go to **Settings → Pages → Source → main branch → / (root)**
4. Your site will be live at `https://<your-username>.github.io/portfolio`

---

## 📝 How to Personalize

1. Replace `assets/images/profile.jpg` with your own photo
2. Replace all certificate images in `assets/certificates/`
3. Replace project screenshots in `assets/projects/`
4. Update your **email, phone, LinkedIn, GitHub links** in `index.html`
5. Update project **GitHub and Live links** in the Projects section
